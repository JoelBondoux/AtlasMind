## Shape Atlas With A Personality Profile

The Personality Profile is a guided workspace dashboard for tuning how Atlas behaves.

Use it to:

1. Define Atlas's role, tone, reasoning posture, and red lines.
2. Leave most prompts on **auto** when the default behavior is already acceptable.
3. Save real workflow defaults like budget mode, speed mode, approval policy, and chat carry-forward limits.
4. Sync the resulting profile into `project_memory/agents/` and a summary block inside `project_soul.md` when SSOT is available.

This is the fastest way to make Atlas feel intentional without manually editing settings and memory files separately.
