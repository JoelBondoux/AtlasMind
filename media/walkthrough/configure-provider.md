## Configure a Model Provider

AtlasMind supports **8 providers**: Anthropic, OpenAI, Google Gemini, Mistral, DeepSeek, z.ai, Local LLM, and GitHub Copilot.

1. Open **AtlasMind: Manage Model Providers** from the Command Palette.
2. Click **Set API Key** next to your preferred provider.
3. Keys are stored securely in VS Code SecretStorage — never on disk.

GitHub Copilot works with your existing VS Code sign-in — no extra key needed.
