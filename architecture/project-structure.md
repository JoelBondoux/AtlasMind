## Architecture

The project is structured as follows:

- `.atlasmind/`: AtlasMind-specific configuration and data.
- `.claude/`: Claude-specific configuration and data.
- `.continue/`: Continue-specific configuration and data.
- `.git/`: Git repository metadata.
- `.github/`: GitHub-specific configuration (e.g., workflows, issue templates).
- `.vscode/`: VS Code-specific configuration.
- `architecture/`: Architecture documentation and diagrams.
- `coverage/`: Test coverage reports.
- `docs/`: Documentation.
- `src/`: Source code.
- `test/`: Tests.
- `wiki/`: Wiki documentation.

## Key Files

- `CHANGELOG.md`: Changelog.
- `LICENSE`: License information.
- `README.md`: Readme.
- `SECURITY.md`: Security policy.
- `package.json`: Package manifest.
- `tsconfig.json`: TypeScript configuration.
